# 🌼 React To-Do List

A cute and simple To-Do List application built using React.js 💛  
Users can add tasks ✍️ and remove them by clicking on the task item ❌

---

## ✨ Features

- ➕ Add new tasks
- 🗑️ Delete tasks by clicking on them
- 🎨 Cute and minimal UI
- ⚛️ Built using React Hooks (`useState`)
- 🧩 Component-based structure

## 🚀 How It Works

### ➕ Adding a Task
- User types a task into the input field
- Clicking the **Add** button adds the task to the list

### ❌ Deleting a Task
- Clicking on a task removes it from the list

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/react-todo-list.git
```

Move into the project folder:

```bash
cd react-todo-list
```

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm start
```
